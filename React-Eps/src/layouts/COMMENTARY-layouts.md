# Comentario de layouts

## Proposito
Layouts que definen estructura visual y navegacion por modulo/rol.

## Layouts principales
- `AdminLayout.jsx`: shell principal de administracion.
- `DoctorLayout.jsx`: shell del profesional medico.
- `PatientLayout.jsx`: experiencia de autoservicio del paciente.
- `FarmaciaLayout.jsx`: operacion farmaceutica.
- `SuperAdminLayout.jsx`: gestion global de plataforma.
- `LoginLayout.jsx` e `IndexLayout.jsx`: rutas publicas y autenticacion.

## Componentes de soporte
- Carpeta `Layout-Components/`: headers, sidebars, dropdowns e items de menu.
