# Comentario de hooks

## Proposito
Hooks personalizados para encapsular consumo API, estado y transformaciones de datos.

## Grupos funcionales
- **Catalogos**: estados, roles, ciudades, departamentos, especialidades, tipos y categorias.
- **Usuarios**: medicos, usuarios por rol, tablas y filtros.
- **Clinica**: citas, historial, medicos disponibles y tipos de cita.
- **Reportes**: carga de indicadores y consultas analiticas.
- **Soporte UI**: ayuda contextual y tema.
