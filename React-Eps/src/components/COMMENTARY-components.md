# Comentario de componentes

## Proposito
Componentes reutilizables del frontend.

## Secciones
- `Routes/`: guardias de rutas por autenticacion y rol.
- `Login/`: piezas de flujo de acceso y recuperacion.
- `Users/`: tablas y bloques visuales de usuarios.
- `UI/`: modales, toasts, inputs y bloques de interfaz reutilizables.
- Modulos especializados (`Citas/`, `DoctorSchedule/`, `Reports/`, etc.) para funciones concretas.
